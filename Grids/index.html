What is CSS Grid?

CSS Grid Layout is a powerful 2-dimensional layout system for the web. It lets you create complex web layouts with rows and columns, making it much easier to design responsive pages without using floats or positioning.

Key Grid Concepts

Grid Container & Items

The element with display: grid becomes a grid container. Its direct children automatically become grid items.

Grid Lines

The dividing lines that make up the grid structure. They are numbered starting from 1, or from the end with negative numbers.

Grid Tracks

The space between two adjacent grid lines (rows or columns). You define these with grid-template-rows and grid-template-columns.

Grid Cells & Areas

A grid cell is a single unit. A grid area is a rectangular area made up of multiple cells, defined by four grid lines.

fr Unit

The fractional unit (fr) represents a fraction of the available space in the grid container. It distributes space proportionally.

Implicit vs Explicit Grid

The explicit grid is defined with template properties. The implicit grid is created automatically when items are placed outside the explicit grid.
